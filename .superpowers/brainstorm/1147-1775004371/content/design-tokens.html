文字层级（对比度全部 ≥ 4.5:1）

--text-primary #e6edf3 ← 主文字（对比 14.5:1）
--text-secondary #8b949e ← 次要文字（对比 5.9:1）✓
--text-muted #6e7681 ← 时间戳/标签（对比 4.5:1）✓
--text-link #58a6ff ← 链接/强调
--text-success #3fb950 ← 成功/工具完成

⚠ 当前 --muted: #7b6857 在 #f5f0e8 背景上对比仅 3.8:1（不达标），修复后全部 ≥ 4.5:1

语义强调色

success / idle
工具成功、会话空闲、文件写入
warning / running
Agent 运行中、待审权限
error / danger
工具失败、权限拒绝、错误消息
info / explore mode
explore/spec 模式、信息类工具
accent / plan mode
spec/plan 内容、权限上下文
diff added
+新增行 / diff removed 删除行
